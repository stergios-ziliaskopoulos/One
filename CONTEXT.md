# Project Goal
Θέλουμε να φτιάξουμε ένα απλό script σε Python.
Στόχος: Να παίρνει την τρέχουσα τιμή του Bitcoin από ένα δημόσιο API και να μου το τυπώνει στην οθόνη.
Απαιτήσεις:
- Χρήση της βιβλιοθήκης `requests`.
- Χρήση του CoinGecko API.
- Error handling (αν δεν δουλεύει το API, να μου πει γιατί).
